# Isaiah Decoded

An interactive study tool for understanding Isaiah through Avraham Gileadi's typological framework.

## Features

- **Full Concordance** — Click any word to find every occurrence across all 66 chapters
- **Code Word Decoder** — Toggle decoded view to see symbolic meanings (Babylon = worldliness, etc.)
- **7-Rung Ladder** — Gileadi's spiritual categories with icons and self-reflection questions
- **Chiastic Structure** — See each chapter's position in Isaiah's bifid design (1-33 mirrors 34-66)
- **Cross-References** — 506 cross-references with full verse text from all standard works
- **Highlighting & Underlining** — 12 colors (7 rung + 5 custom) with word-snap precision
- **Word Grouping** — Create named groups with visual connecting lines
- **1 Nephi 20-21 Master Key** — Verse-by-verse walkthrough with annotations
- **Kings & Queens** — Christ's parables reframed as royal instruction
- **The Ascent** — Three parallel paths (Covenant Path, Beatitudes, D&C 4:6)
- **Personal Notes** — Record your own keys and insights
- **Dark Mode** — Full dark mode support
- **Fully Offline** — Works without internet after first load
